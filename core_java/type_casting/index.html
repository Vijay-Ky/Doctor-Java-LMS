Type Conversion in Java

Type conversion (also known as type casting) in Java is the process of converting a value from one data type to another. It can be categorized into two types: Widening (Implicit) and Narrowing (Explicit) conversions.

Types of Type Conversion

  1. Widening Type Conversion (Implicit): Converting a smaller data type to a larger one. It is safe and done automatically by Java.
  2. Narrowing Type Conversion (Explicit): Converting a larger data type into a smaller one. It may result in data loss and must be done manually by the programmer using casting.

Widening (Implicit) Conversion

Java performs this type of conversion automatically. No loss of information occurs. 

byte → short → int → long → float → double
  
public class WideningExample {
    public static void main(String[] args) {
        int i = 100;
        long l = i; // int to long
        float f = l; // long to float
        System.out.println(i);
        System.out.println(l);
        System.out.println(f);
    }
}

Narrowing (Explicit) Conversion

Requires explicit casting by the programmer. It may cause data loss or truncation. 

double → float → long → int → short → byte
  
public class NarrowingExample {
    public static void main(String[] args) {
        double d = 123.456;
        long l = (long) d; // Explicit casting
        int i = (int) l;
        System.out.println(d);
        System.out.println(l);
        System.out.println(i);
    }
}

Type Casting with Objects (Upcasting & Downcasting)

Type conversion can also happen with objects in the inheritance hierarchy.

  • Upcasting: Automatically converting subclass type to superclass type.
  • Downcasting: Manually converting superclass type back to subclass. Requires explicit cast and should be done with care.
class Animal {}
class Dog extends Animal {
    void bark() {
        System.out.println("Barking...");
    }
}

public class ObjectCasting {
    public static void main(String[] args) {
        Animal a = new Dog(); // Upcasting
        Dog d = (Dog) a;      // Downcasting
        d.bark();
    }
}

Invalid Type Conversion

Java does not allow type conversion between incompatible types (e.g., converting `int[]` to `byte[]` directly). 

public class InvalidConversion {
    public static void main(String[] args) {
        int[] a = new int[5];
        byte[] b = new byte[5];
        // a = b; // ❌ Compile-time error
    }
}
